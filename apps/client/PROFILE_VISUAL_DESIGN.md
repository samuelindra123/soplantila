## Design Tokens

### Colors (Light Mode)
- **Cover Gradient**: `from-accent/20 via-accent/10 to-surface-dark`
- **Background**: `#ffffff`
- **Foreground**: `#111111`
- **Accent**: `#0071e3` (Apple Blue)
- **Muted**: `#6e6e73`
- **Border**: `rgba(0, 0, 0, 0.06)`

### Colors (Dark Mode)
- **Cover Gradient**: `from-accent/20 via-accent/10 to-surface-dark`
- **Background**: `#000000`
- **Foreground**: `#f5f5f7`
- **Accent**: `#0a84ff`
- **Muted**: `#8e8e93`
- **Border**: `rgba(255, 255, 255, 0.1)`

### Spacing
- **Avatar Desktop**: `160x160px` (w-40 h-40)
- **Avatar Mobile**: `128x128px` (w-32 h-32)
- **Cover Desktop**: `320px` (h-80)
- **Cover Tablet**: `256px` (h-64)
- **Cover Mobile**: `192px` (h-48)
- **Border Radius**: `2.5rem` (40px) untuk cards
- **Border Radius**: `2rem` (32px) untuk avatar

### Typography
- **Name Desktop**: `text-3xl lg:text-4xl` (30px - 36px)
- **Name Mobile**: `text-2xl` (24px)
- **Username**: `text-sm sm:text-base` (14px - 16px)
- **Bio**: `text-[15px] sm:text-base` (15px - 16px)
- **Metadata**: `text-sm` (14px)

### Animations
- **Hover Scale**: `hover:scale-105` untuk avatar
- **Button Scale**: `active:scale-95`
- **Stagger Delay**: `${i * 100}ms` untuk post cards
- **Transition**: `duration-300` - `duration-500`

## Component Hierarchy

```
ProfilePage
├── Sidebar (global)
└── main
    ├── ProfileHeader
    │   ├── Cover Section
    │   ├── Avatar + Name + Actions
    │   ├── Stats (Followers, Following, Posts)
    │   ├── Bio
    │   └── Metadata (Job, Location, Birthday)
    ├── ProfileTabs
    │   └── Tabs: Posts | Media | About | Likes
    └── Tab Content
        ├── ProfilePosts (tab: posts)
        │   ├── Empty State (if no posts)
        │   └── PostCard[] (if has posts)
        └── ProfileEmptyTab (tabs: media, about, likes)
```

---

## Interactive States

### Tabs
- **Default**: `text-muted`
- **Hover**: `hover:text-foreground`
- **Active**: `text-foreground` + accent underline
- **Focus**: `focus-visible:outline`

### Buttons
- **Edit Profile**:
  - Default: `bg-surface-dark border border-border-soft`
  - Hover: `bg-surface border-accent/30 shadow-sm`
  - Active: `scale-95`

- **Follow**:
  - Default: `bg-accent text-white`
  - Hover: `bg-accent-strong shadow-sm`
  - Active: `scale-95`

### Avatar
- **Default**: `shadow-premium`
- **Hover**: `scale-105 shadow-md` (smooth transition)

- ✅ Large cover photo
- ✅ Prominent profile picture overlapping cover
- ✅ Stats row (followers, following, posts)
- ✅ Bio section
- ✅ Tab navigation
- ✅ Content feed below tabs
- ✅ Clean, minimal aesthetic
- ✅ Premium glassmorphism effects
- ✅ Smooth transitions
- ✅ Rounded corners everywhere
